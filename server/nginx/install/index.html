    安装教程

    四项确认

    • 确认系统网络（ping）
    • 确认 yum 可用（yum list | grep gcc）
    • 确定关闭 iptables（iptables -F）
    • 确认停用 selinux

    两项安装

    yum -y install gcc gcc-c++ autoconf pcre pcre-devel make automake
    yum -y install wget httpd-tools vim

    安装源

    cd /etc/yum.repos.d
    vim nginx.repo
    # 添加
    [nginx]
    name=nginx repo
    baseurl=http://nginx.org/packages/centos/7/$basearch/
    gpgcheck=0
    enabled=1

    进行安装

    yum list | grep nginx
    yum install nginx

    查看版本

    查看版本

    nginx -v
